## Introduction

opspec provides a set of vendor & platform agnostic standards for
defining, distributing, and controlling operations.

### Roadmap

The long term vision being worked towards includes:

#### Scope

- op definition format
- op control API
- op data API

#### Deliverables

- IETF RFC(s)

## Contributing
